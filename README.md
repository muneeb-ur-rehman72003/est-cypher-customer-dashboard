# Music Platform Customer Dashboard Frontend

A responsive customer dashboard frontend built using HTML5, CSS3, and JavaScript.
The dashboard is designed for a music platform and includes user-focused sections, modern dark UI styling, dashboard cards, navigation panels, content blocks, and organized frontend layouts.
